# GhidraGPT

## My notes
This is a cool Ghidra plugin that uses LLMs inside reverse engineering. I added it because it shows how AI can help with binary analysis and finding security issues in decompiled code.

## Why I keep it here
It is useful for learning how to combine AI with reverse engineering tools like Ghidra.

## What it does
- Adds model-based code explanation and analysis to Ghidra
- Helps rename functions and variables for readability
- Can flag potential vulnerabilities in decompiled code
- Works with many model providers like OpenAI, Claude, Gemini, and others

## When I use it
- When I want AI help during malware or binary analysis
- When I need faster understanding of decompiled functions
- When I want to see how AI integrates into security tooling
