# Meta SecAlign

## My notes
This repo is about building a foundation model with better prompt injection defenses. It is research from Facebook AI Research and I keep it here because it is one of the best current examples of secure LLM design.

## Why it matters
If I want to understand how to protect models from malicious prompts, this is the reference project.

## What it shows
- Models trained to resist prompt injection
- How to evaluate prompt injection defenses across benchmarks
- A security vs utility tradeoff for LLMs
- Support for Meta-SecAlign-8B and 70B defended models

## When I use it
- When studying safe model training
- When comparing prompt injection defenses
- When I want a real example of research-grade LLM security
