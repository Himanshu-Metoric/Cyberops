# Zero Trust Quick Checklist

- [ ] Inventory all assets and classify by sensitivity
- [ ] Enforce MFA and SSO for all users
- [ ] Implement device posture checks (EDR, patch level)
- [ ] Define least-privilege roles and access groups
- [ ] Create microsegments for critical assets
- [ ] Implement policy engine (ZTNA/CASB) for access enforcement
- [ ] Monitor anomalous access with UEBA and logging
- [ ] Run tabletop exercises for assume-breach scenarios
- [ ] Educate users on phishing and credential hygiene
